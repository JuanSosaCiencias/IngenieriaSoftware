# Módulos

* Cátalogo

Sección para ver el catálogo de las **becas**, cursos, concursos, etc. Dentro del catalogo de becas contamos con un filtro para poder explorar categorías específicas al interés del usuario. El catalogo se compone de **tarjetas informativas** donde aparece la convocatoria de cada programa en el se encuentra información resumida como requisitos, fecha límite, beneficios, nombre de la institución que la ofrece, modalidad ( en linea/presencial), duración, foto del programa. Además la tarjeta cuanta con un botón de *Ver Convocatoria*, un botón de *Foro* y un botón de *Guardar*.

* Detalles de la convocatoria

Esta sección es una pagina donde se muestra toda la información de la convocatoria. Y se encuentra un boton para aplicar, además al final de la pagina se encuentra la sección de foro de la beca.

* Foro

Este módulo es un foro de discusión que simula una sección de comentarios donde los usuarios pueden preguntar cosas y también ver si hay participantes que estén interesados en colaborar.

* Login ( Usuario/ Organización)

Sección para entrar la plataforma. Ver contenido de interés al perfil, ver y modificar datos personales.

* Registro (Usuario/ Organizacion)

Sección para poder crear un **usuario**.
  
* Registro para Organizaciones

Aquí se pueden registrar las organizaciones, perfiles, o afiliaciones que quieran ser difusoras de contenido. Para esto
además de incluir otros datos como Descripcion de la organización, se necesita verificar la cuenta para poder otorgar los permisos de difusión.

* Crear Curriculum.

En esta sección los usuarios podrán escribir sus experiencias, grados de estudio, y actividades que han realizado. Así como
áreas de interés. Los datos de contacto cuentan con un parámetro de visibilidad que le permite al usuario la posibilidad de manejar la privacidad de sus datos.

* Curriculum

Se muestra el perfil de los usuarios, en el se encuentra un botón para poder hacer amigos con otras personas.

* Becas guardadas

Se guardan todas las tarjetas que el usuario marco como guardadas, y incluye un marcador de status (revisada, solicitada, en espera, recibida/rechazada). Si el marcador todavía no se encuentra como solicita aparece un marcador con el tiempo restante para aplicar. Si la convocatoria es recibida entonces se incorpora al curriculum de los perfiles.

* Filtro de búsqueda

Aquí es posible buscar convocatorias, perfiles, organizaciones.
